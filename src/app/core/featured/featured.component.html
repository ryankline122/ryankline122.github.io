Web-based Multiplayer Board Game Demo

The motivation behind this project was not necessarily to build a complete game, rather I wanted to understand what exactly goes into making one. Things like figuring out how to move pawns around a board, implementing turn-based logic, or setting up an in-game lobby.

Built with Pixi.js and Golang, this project provided me with the opportunity to work extensively with the HTML canvas API on the front-end and websockets on the back.

Super Smash Bros. Melee Bot

Using an expert-system style approach, I created an AI agent to play as Donkey Kong in Super Smash Bros. Melee. A Python script reads in directly from the games memory to access important gamestate like player location, health, and remaining lives to make informed decisions on what actions to take at any given moment.

This project was completed as part of a semester long project in Grand Valley's Applied Artificial Inteligence course and was built with Python and by using a fork of the Dolphin Emulator.

CLI Connect 4 with Elixir

Connect 4 has always been my favorite game to program when learning a new language. It's simple enough, but exposes you to several key language features such as dealing with user input, working with 2D arrays and unit testing for example.

I chose Elixir this time around since I wanted to give functional programming another shot after hearing great things about it online. The only other time I used this programming paradigm was in my junior year of college where I clawed my way through the same project using Kawa (a dialect of Scheme). Elixir was much more enjoyable and gave me a greater appreciation for functional programming.
